Logging facility

We keep lazily the last N messages of every log levels. Additionally, messages of higher level than some threshold are copied onto stderr.

type level =
| Critical
| Error
| Warning
| Info
| Debug 
type msg = Clock.Time.t * string Lazy.t 
type queue = int * msg array 
type logger = {
   name :string;
   queues :queue array;
}
